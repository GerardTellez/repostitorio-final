# Ordres Principals de Git

1. **git init**
   - Inicialitza un repositori nou en la carpeta actual.
   
2. **git status**
   - Mostra l'estat dels fitxers (modificats, nous, o llestos per fer commit).
   
3. **git add <fitxer>**
   - Afegeix fitxers a l'àrea de preparació (staging area).
   
4. **git commit -m "missatge"**
   - Guarda els canvis de l'àrea de preparació a l'historial amb un missatge descriptiu.
   
5. **git push origin <branca>**
   - Puja els canvis locals al repositori remot (GitHub).
